# MECTECH

Projeto em Next.js criado seguindo design figma

## Tecnologias Usadas

- **Next.js**
- **Tailwind CSS**
- **Zustand** 
- **TypeScript** 

## Como Iniciar o Projeto

1. **Clone o repositório**

   ```bash
   git clone https://github.com/restlucas/mectech
   cd mectech
   ```

2. **Instale as dependências**
   ```bash
   npm install
   ```
